# 开心消消乐
使用cocos creator 编写的三消游戏。
这些随机性，就是玩家一次又一次赌自己的运气，当运气好时，连续combo次数多，就会让玩家很爽；但combo少，玩家也只会觉得运气不好，而不是我的操作有问题，所以，对于失败的接受性就会更容易。
